死了么 App 原型设计

专为独居90后年轻人设计的养老App，每天问一句，防止独自死在家里没人发现。融合了幽默网络文化和实用功能，让孤独生活更安心。

原型导航说明

  • 所有界面均按照 iPhone 15 Pro 尺寸设计 (375×812px)
  • 点击底部导航栏可以在各个页面间切换
  • UI 设计风格为简约清新，搭配网络流行语，符合年轻人审美
  • 所有界面使用 HTML + Tailwind CSS 构建，可直接用于开发
登录/注册界面
用户通过手机号快速注册登录，支持社交账号一键登入
首页（打卡界面）
每日打卡、健康状态和紧急求助一目了然
健康监测页面
记录并追踪健康数据，提供生活小贴士
骑手服务页面
提供上门探视、日常服务和紧急救援
社区页面
独居青年的交流平台，分享生活趣事与孤独经验
个人中心
个人资料、紧急联系人和应用设置管理
设置页面
隐私设置、通知管理和个性化配置
